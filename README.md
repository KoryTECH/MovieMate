# MovieMate

A sleek and intuitive movie search application that fetches comprehensive movie information from the OMDB API (IMDb database).

## Features

- **Movie Search**: Search for any Hollywood movie by title
- **Detailed Information**: Displays the following movie details:
  - Movie title
  - Genre
  - Director
  - Cast/Actors
  - Plot summary
  - Release year
  - IMDb rating
  - Movie poster image
- **Clean UI**: Simple, user-friendly interface with a vibrant blue theme
- **Real-time Results**: Instant search results from IMDb database

## Demo

The application features a centered search interface with a bright blue background, making it easy to focus on movie searches.

## Technologies Used

- HTML5
- CSS3
- JavaScript
- OMDB API (IMDb data source)

## Usage

1. Enter a movie name in the search field
2. Click the "Search" button
3. View detailed information about the movie including poster, plot, cast, and ratings

## API Information

This project uses the [OMDB API](http://www.omdbapi.com/) to fetch movie data from IMDb. The OMDB API provides detailed information about movies, TV shows, and more.
